# vuetify-calendar

基于[dayspan-vuetify](https://github.com/ClickerMonkey/dayspan-vuetify)的demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
